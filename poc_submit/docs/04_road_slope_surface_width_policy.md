# 도로 경사·노면·폭 표시 정책

도로 레이어는 경로 추천 확정 로직이 아니라 PoC 시각화 및 기준 검토용이다. 보행 등급은 아직 확정하지 않고, 현재 지도에서는 경사 구간, 노면 선 모양, 폭 5단계만 보여준다.

## 경사 구간

| 등급 | 범위 | 표시 |
|---|---|---|
| `VERY_GENTLE` | 0~3% | 매우 완만 |
| `GENTLE` | 3~5% | 완만 |
| `CAUTION` | 5~8% | 주의 |
| `SLOPED` | 8~12% | 경사 |
| `STEEP` | 12~20% | 급경사 |
| `VERY_STEEP` | 20~35% | 매우 급경사 |
| `DANGEROUS` | 35%+ | 위험 |
| `UNKNOWN` | 미산출 | 회색 |

## 노면 선 모양

| surfaceType | 표시 |
|---|---|
| `PAVED` | 실선 |
| `UNPAVED` | 파선 |
| `UNKNOWN` | 점선 |

## 폭 5단계

| widthLevel | 기준 |
|---|---|
| `VERY_NARROW` | 2m 미만 |
| `NARROW` | 2~4m |
| `MEDIUM` | 4~8m |
| `WIDE` | 8~10m |
| `VERY_WIDE` | 10m 초과 |
| `UNKNOWN` | 미분류 |

## 성능 정책

도로 데이터는 구군별 JS 파일로 분리하고 기본 OFF로 둔다. 사용자가 구군을 선택하고 도로 레이어를 켤 때만 해당 구군 데이터를 로드한다. 지도 렌더링은 Leaflet Canvas renderer를 사용하고, JS 산출물은 표시와 팝업에 필요한 필드만 남긴다.
