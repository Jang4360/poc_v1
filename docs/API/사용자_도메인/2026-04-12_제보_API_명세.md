## 1. API 목록

| 우선순위 | API | 설명 |
| --- | --- | --- |
| P1 | `POST /hazard-reports` | 도로 상태 제보 등록 |

> 제보 상태 변경(`PENDING` → `APPROVED`/`REJECTED`)은 사용자 API가 아니라 `POST /slack/hazard-reports/actions` 운영용 Slack 콜백 API에서 처리한다.
> 도로 상태 제보는 익명 제보로 처리하며 사용자 계정과 연결하지 않는다. 사용자별 제보 목록/상세/수정/삭제 API는 제공하지 않는다.

---

## 2. 도로 상태 제보 등록

| 항목 | 내용 |
| --- | --- |
| 우선순위 | P1 |
| Method | `POST` |
| URL | `/hazard-reports` |
| 인증 | 불필요 |
| 설명 | 도로 공사, 장애물, 점자블록 손상 등을 익명으로 제보한다. 파일 업로드는 제외하고 이미지 URL 배열만 고려한다. |

### Request

#### Header

| 이름 | 필수 | 설명 |
| --- | --- | --- |
| Content-Type | Y | `application/json` |

#### Body

| 필드명 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| reportType | string | Y | 제보 유형. `CONSTRUCTION`, `OBSTACLE`, `DAMAGE`, `OTHER` |
| description | string | N | 설명 |
| reportPoint | object | Y | 제보 위치 |
| reportPoint.lat | number | Y | 위도 |
| reportPoint.lng | number | Y | 경도 |
| address | string | N | 주소 |
| imageUrls | array | N | 제보 이미지 URL 목록 |

#### Request Example

```json
{
  "reportType": "OBSTACLE",
  "description": "보도 중앙에 이동을 방해하는 적치물이 있습니다.",
  "reportPoint": {
    "lat": 35.1686,
    "lng": 129.0576
  },
  "address": "부산광역시 부산진구 시민공원로 73",
  "imageUrls": [
    "https://example.com/reports/1/image-1.jpg"
  ]
}
```

### Response

- HTTP Status: `201 Created`

| 필드명 | 타입 | Null 가능 | 설명 |
| --- | --- | --- | --- |
| reportId | Long | N | 제보 ID |
| status | string | N | 제보 상태. 기본값 `PENDING` |

#### Success Example

```json
{
  "success": true,
  "data": {
    "reportId": 1,
    "status": "PENDING"
  },
  "message": null
}
```

### Error

| HTTP Status | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 400 | INVALID_INPUT | 잘못된 입력입니다. | 제보 유형 또는 제보 위치 누락/형식 오류 |
