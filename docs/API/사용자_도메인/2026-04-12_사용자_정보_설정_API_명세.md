# 📋 사용자 정보/설정 API 명세

> **작성일:** 2026-04-12  
> **작성자:** 김응서  
> **최종 수정일:** 2026-04-12

---

## 1. API 목록

| 우선순위 | API | 설명 |
| --- | --- | --- |
| P1 | `GET /users/me` | 내 정보 조회 |
| P1 | `PATCH /users/me` | 닉네임, 장애유형, 장애등급, 전화번호 수정 |
| P1 | `PATCH /users/me/settings` | 사용자 설정 수정 |

---

## 2. 내 정보 조회

| 항목 | 내용 |
| --- | --- |
| 우선순위 | P1 |
| Method | `GET` |
| URL | `/users/me` |
| 인증 | 필요 |

### Request

#### Header

| 이름 | 필수 | 설명 |
| --- | --- | --- |
| Authorization | Y | Bearer 액세스 토큰 |

#### Query Parameter

> 없음

#### Body

> 없음

### Response

- HTTP Status: `200 OK`

| 필드명 | 타입 | Null 가능 | 설명 |
| --- | --- | --- | --- |
| userId | string(UUID) | N | 사용자 ID |
| nickname | string | N | 표시용 닉네임 |
| socialProvider | string | N | 소셜 제공자 |
| disabilityType | string | Y | 장애 유형. 프로필 미완료 시 `null` |
| disabilityGrade | string | Y | 장애 등급 |
| phoneNumber | string | N | 전화번호 |
| locationTermsAgreed | boolean | N | 위치 약관 동의 여부 |
| locationTermsAgreedAt | string | Y | 위치 약관 동의 일시 |
| ttsEnabled | boolean | N | 음성 안내 여부 |
| routeCollectionEnabled | boolean | N | 경로 데이터 수집 동의 여부 |
| pushEnabled | boolean | N | 푸시 알림 여부 |
| profileCompleted | boolean | N | 필수 프로필 입력 완료 여부 |

#### 비고

- `disabilityType=null`은 소셜 로그인 직후 필수 프로필을 아직 입력하지 않은 임시 상태에서만 허용한다.
- `profileCompleted=false`인 사용자는 회원가입 완료 상태가 아니며, 클라이언트는 프로필 입력 화면으로 이동시켜야 한다.
- 경로 탐색, 대중교통 후보 조회, 경로 로그 저장 등 서비스 핵심 기능에서는 `disabilityType`이 `VISUAL` 또는 `MOBILITY`로 확정된 사용자만 허용한다.

#### Success Example

```json
{
  "success": true,
  "data": {
    "userId": "018f7f6c-2b7e-7c3a-9f4a-8b4e3b7c9a01",
    "nickname": "응서",
    "socialProvider": "KAKAO",
    "disabilityType": "MOBILITY",
    "disabilityGrade": "3",
    "phoneNumber": "119",
    "locationTermsAgreed": true,
    "locationTermsAgreedAt": "2026-04-12T17:00:00+09:00",
    "ttsEnabled": true,
    "routeCollectionEnabled": false,
    "pushEnabled": true,
    "profileCompleted": true
  },
  "message": null
}
```

### Error

| HTTP Status | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 401 | UNAUTHORIZED | 인증이 필요합니다. | 토큰 누락/만료 |
| 404 | RESOURCE_NOT_FOUND | 리소스를 찾을 수 없습니다. | 사용자 정보가 존재하지 않음 |

## 3. 내 정보 수정

| 항목 | 내용 |
| --- | --- |
| 우선순위 | P1 |
| Method | `PATCH` |
| URL | `/users/me` |
| 인증 | 필요 |
| 설명 | 신규 사용자의 필수 프로필 입력 또는 마이페이지의 기본 정보 수정을 처리한다. 신규 사용자는 `nickname`, `disabilityType`을 입력해야 회원가입 완료 상태가 된다. 닉네임은 중복을 허용한다. |

### Request

#### Header

| 이름 | 필수 | 설명 |
| --- | --- | --- |
| Authorization | Y | Bearer 액세스 토큰 |
| Content-Type | Y | `application/json` |

#### Body

| 필드명 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| nickname | string | N | 표시용 닉네임. 중복 허용 |
| disabilityType | string | N | 장애 유형. `VISUAL`, `MOBILITY` |
| disabilityGrade | string | N | 장애 등급 |
| phoneNumber | string | N | 전화번호 |

#### 비고

- 최초 소셜 로그인 사용자는 `nickname`, `disabilityType`을 입력해야 회원가입 완료 상태(`profileCompleted=true`)가 된다.
- `profileCompleted=false` 상태에서는 지도/경로 탐색 등 서비스 핵심 기능을 이용할 수 없다.
- 기존 프로필 수정 사용자는 변경할 필드만 전달할 수 있다.

#### Request Example

```json
{
  "nickname": "응서",
  "disabilityType": "MOBILITY",
  "disabilityGrade": "3",
  "phoneNumber": "119"
}
```

### Response

- HTTP Status: `200 OK`

| 필드명 | 타입 | Null 가능 | 설명 |
| --- | --- | --- | --- |
| userId | string(UUID) | N | 사용자 ID |
| nickname | string | N | 변경된 닉네임 |
| disabilityType | string | N | 변경된 장애 유형 |
| disabilityGrade | string | Y | 변경된 장애 등급 |
| phoneNumber | string | N | 변경된 전화번호 |
| profileCompleted | boolean | N | 필수 프로필 입력 완료 여부 |

#### Success Example

```json
{
  "success": true,
  "data": {
    "userId": "018f7f6c-2b7e-7c3a-9f4a-8b4e3b7c9a01",
    "nickname": "응서",
    "disabilityType": "MOBILITY",
    "disabilityGrade": "3",
    "phoneNumber": "119",
    "profileCompleted": true
  },
  "message": null
}
```

### Error

| HTTP Status | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 400 | INVALID_INPUT | 잘못된 입력입니다. | `nickname`, `disabilityType`, `disabilityGrade`, `phoneNumber` 형식 오류 또는 프로필 필수값 누락 |
| 401 | UNAUTHORIZED | 인증이 필요합니다. | 토큰 누락/만료 |
| 404 | RESOURCE_NOT_FOUND | 리소스를 찾을 수 없습니다. | 사용자 정보가 존재하지 않음 |

## 4. 사용자 설정 수정

| 항목 | 내용 |
| --- | --- |
| 우선순위 | P1 |
| Method | `PATCH` |
| URL | `/users/me/settings` |
| 인증 | 필요 |
| 설명 | 위치 약관 동의, TTS, 경로 데이터 수집 동의, 푸시 알림 설정을 한 번에 수정한다. |

### Request

#### Header

| 이름 | 필수 | 설명 |
| --- | --- | --- |

#### Body

| 필드명 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| locationTermsAgreed | boolean | N | 위치 약관 동의 여부 |
| ttsEnabled | boolean | N | 음성 안내 여부 |
| routeCollectionEnabled | boolean | N | 경로 데이터 수집 동의 여부 |
| pushEnabled | boolean | N | 푸시 알림 여부 |

#### Request Example

```json
{
  "locationTermsAgreed": true,
  "ttsEnabled": true,
  "routeCollectionEnabled": false,
  "pushEnabled": true
}
```

### Response

- HTTP Status: `200 OK`

| 필드명 | 타입 | Null 가능 | 설명 |
| --- | --- | --- | --- |
| locationTermsAgreed | boolean | N | 위치 약관 동의 여부 |
| locationTermsAgreedAt | string | Y | 위치 약관 동의 일시 |
| ttsEnabled | boolean | N | 음성 안내 여부 |
| routeCollectionEnabled | boolean | N | 경로 데이터 수집 동의 여부 |
| pushEnabled | boolean | N | 푸시 알림 여부 |

#### Success Example

```json
{
  "success": true,
  "data": {
    "locationTermsAgreed": true,
    "locationTermsAgreedAt": "2026-04-12T17:00:00+09:00",
    "ttsEnabled": true,
    "routeCollectionEnabled": false,

  },
  "message": null
}
```

### Error
