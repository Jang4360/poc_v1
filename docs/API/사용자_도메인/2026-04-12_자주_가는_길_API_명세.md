# 📋 자주 가는 길 API 명세

> **작성일:** 2026-04-12  
> **작성자:** 김응서  
> **최종 수정일:** 2026-04-12

---

## 1. API 목록

| 우선순위 | API | 설명 |
| --- | --- | --- |
| P1 | `GET /favorite-routes` | 자주 가는 길 목록 조회 |
| P1 | `POST /favorite-routes` | 자주 가는 길 저장 |
| P1 | `PATCH /favorite-routes/{favRouteId}` | 자주 가는 길 수정 |
| P1 | `DELETE /favorite-routes/{favRouteId}` | 자주 가는 길 삭제 |

> 자주 가는 길 서버 API는 로그인 사용자의 서버 저장 데이터만 처리한다. 비로그인 사용자는 이 API를 호출하지 않고 앱 로컬 저장소에서 처리한다.

---

## 2. 자주 가는 길 목록 조회

| 항목 | 내용 |
| --- | --- |
| 우선순위 | P1 |
| Method | `GET` |
| URL | `/favorite-routes` |
| 인증 | 필요 |
| 설명 | 로그인 사용자가 서버에 저장한 자주 가는 길 목록을 조회한다. 비로그인 사용자의 자주 가는 길은 클라이언트 로컬 저장소에서 관리한다. |

### Request

#### Header

| 이름 | 필수 | 설명 |
| --- | --- | --- |
| Authorization | Y | Bearer 액세스 토큰 |

#### Query Parameter

| 변수명 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| page | int | N | 페이지 번호 |
| size | int | N | 페이지 크기 |

#### Body

> 없음

### Response

- HTTP Status: `200 OK`

| 필드명 | 타입 | Null 가능 | 설명 |
| --- | --- | --- | --- |
| content | array | N | 자주 가는 길 목록 |
| content[].favRouteId | Long | N | 자주 가는 길 ID |
| content[].routeName | string | N | 경로명 |
| content[].startLabel | string | N | 출발지명 |
| content[].endLabel | string | N | 도착지명 |
| content[].startPoint | object | N | 출발지 좌표 |
| content[].endPoint | object | N | 도착지 좌표 |
| content[].routeOption | string | N | 경로 옵션. `SHORTEST`, `SAFE` |
| page | int | N | 현재 페이지 번호 |
| size | int | N | 페이지 크기 |
| totalElements | Long | N | 전체 데이터 수 |
| totalPages | int | N | 전체 페이지 수 |
| hasNext | boolean | N | 다음 페이지 존재 여부 |

#### Success Example

```json
{
  "success": true,
  "data": {
    "content": [
      {
        "favRouteId": 1,
        "routeName": "집에서 병원",
        "startLabel": "부산시민공원",
        "endLabel": "부산역",
        "startPoint": {
          "lat": 35.1686,
          "lng": 129.0576
        },
        "endPoint": {
          "lat": 35.1152,
          "lng": 129.0422
        },
        "routeOption": "SAFE"
      }
    ],
    "page": 0,
    "size": 10,
    "totalElements": 1,
    "totalPages": 1,
    "hasNext": false
  },
  "message": null
}
```

### Error

| HTTP Status | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 401 | UNAUTHORIZED | 인증이 필요합니다. | 토큰 누락/만료 |

## 3. 자주 가는 길 저장

| 항목 | 내용 |
| --- | --- |
| 우선순위 | P1 |
| Method | `POST` |
| URL | `/favorite-routes` |
| 인증 | 필요 |
| 설명 | 로그인 사용자의 자주 가는 길을 서버에 저장한다. 실제 edge 목록이 아니라 재탐색에 필요한 출발지/도착지/옵션을 저장한다. 비로그인 사용자의 자주 가는 길은 클라이언트 로컬 저장소에 저장한다. |

### Request

#### Header

| 이름 | 필수 | 설명 |
| --- | --- | --- |
| Authorization | Y | Bearer 액세스 토큰 |
| Content-Type | Y | `application/json` |

#### Body

| 필드명 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| routeName | string | Y | 경로명 |
| startLabel | string | Y | 출발지명 |
| endLabel | string | Y | 도착지명 |
| startPoint | object | Y | 출발지 좌표 |
| startPoint.lat | number | Y | 위도 |
| startPoint.lng | number | Y | 경도 |
| endPoint | object | Y | 도착지 좌표 |
| endPoint.lat | number | Y | 위도 |
| endPoint.lng | number | Y | 경도 |
| routeOption | string | Y | 경로 옵션. `SHORTEST`, `SAFE` |

#### Request Example

```json
{
  "routeName": "집에서 병원",
  "startLabel": "부산시민공원",
  "endLabel": "부산역",
  "startPoint": {
    "lat": 35.1686,
    "lng": 129.0576
  },
  "endPoint": {
    "lat": 35.1152,
    "lng": 129.0422
  },
  "routeOption": "SAFE"
}
```

### Response

- HTTP Status: `201 Created`

| 필드명 | 타입 | Null 가능 | 설명 |
| --- | --- | --- | --- |
| favRouteId | Long | N | 자주 가는 길 ID |

#### Success Example

```json
{
  "success": true,
  "data": {
    "favRouteId": 1
  },
  "message": null
}
```

### Error

| HTTP Status | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 400 | INVALID_INPUT | 잘못된 입력입니다. | 필수값 누락 또는 좌표/경로 옵션 형식 오류 |
| 401 | UNAUTHORIZED | 인증이 필요합니다. | 토큰 누락/만료 |
