# 📋 장소 검색 API 명세

> **작성일:** 2026-04-12  
> **작성자:** 김응서  
> **최종 수정일:** 2026-04-12

---

## 1. API 목록

| 우선순위 | API | 설명 |
| --- | --- | --- |
| P1 | `GET /places/search` | 텍스트 장소 검색 |
| P2 | `POST /places/search/voice` | 음성 검색 STT |

---

## 2. 텍스트 장소 검색

| 항목 | 내용 |
| --- | --- |
| 우선순위 | P1 |
| Method | `GET` |
| URL | `/places/search` |
| 인증 | 불필요 |
| 설명 | 키워드로 장소를 검색한다. 카카오 API 결과와 내부 `places.providerPlaceId`를 매칭하여 접근성 정보를 함께 제공한다. |

### Request

#### Header

| 이름 | 필수 | 설명 |
| --- | --- | --- |
| Content-Type | Y | `application/json` |

#### Query Parameter

| 변수명 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| keyword | string | Y | 검색어 |
| lat | double | N | 기준 위도 |
| lng | double | N | 기준 경도 |
| radius | int | N | 검색 반경. meter 단위 |
| page | int | N | 페이지 번호 |
| size | int | N | 페이지 크기 |

#### Body

> 없음

### Response

- HTTP Status: `200 OK`

| 필드명 | 타입 | Null 가능 | 설명 |
| --- | --- | --- | --- |
| places | array | N | 검색 결과 목록 |
| places[].placeId | Long | Y | 내부 장소 ID. 매칭 실패 시 `null` |
| places[].providerPlaceId | string | N | 카카오 장소 ID |
| places[].name | string | N | 장소명 |
| places[].category | string | Y | 카테고리 |
| places[].address | string | Y | 주소 |
| places[].distanceMeter | int | Y | 기준 좌표로부터 거리 |
| places[].point | object | N | 좌표 |
| places[].accessibilityFeatures | array | N | 접근성 속성 목록. 내부 장소 매칭 실패 또는 접근성 정보가 없으면 빈 배열 `[]` |
| places[].accessibilityFeatures[].featureType | string | N | 접근성 속성 유형 |
| places[].accessibilityFeatures[].available | boolean | N | 제공 여부 |
| places[].matched | boolean | N | 내부 장소 매칭 여부 |
| page | int | N | 현재 페이지 번호 |
| size | int | N | 페이지 크기 |
| totalElements | Long | N | 전체 데이터 수 |
| totalPages | int | N | 전체 페이지 수 |
| hasNext | boolean | N | 다음 페이지 존재 여부 |

#### Success Example

```json
{
  "success": true,
  "data": {
    "places": [
      {
        "placeId": 10,
        "providerPlaceId": "123456789",
        "name": "부산시민공원",
        "category": "TOURIST_SPOT",
        "address": "부산광역시 부산진구 시민공원로 73",
        "distanceMeter": 350,
        "point": {
          "lat": 35.1686,
          "lng": 129.0576
        },
        "accessibilityFeatures": [
          {
            "featureType": "ramp",
            "available": true
          }
        ],
        "matched": true
      }
    ],
    "page": 0,
    "size": 10,
    "totalElements": 1,
    "totalPages": 1,
    "hasNext": false
  },
  "message": null
}
```

### Error

| HTTP Status | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 400 | INVALID_INPUT | 잘못된 입력입니다. | `keyword` 누락 또는 빈 문자열 |
| 502 | EXTERNAL_API_ERROR | 외부 API 호출에 실패했습니다. | 카카오 API 호출 실패 |

## 3. 음성 검색 STT

| 항목 | 내용 |
| --- | --- |
| 우선순위 | P2 |
| Method | `POST` |
| URL | `/places/search/voice` |
| 인증 | 불필요 |
| 설명 | 클라이언트에서 STT 처리한 음성 인식 결과 텍스트로 장소를 검색하는 후순위 API다. 음성 파일 업로드 방식은 제외한다. |

### Request

#### Header

| 이름 | 필수 | 설명 |
| --- | --- | --- |
| Content-Type | Y | `application/json` |

#### Body

| 필드명 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| recognizedText | string | Y | 클라이언트 STT 또는 외부 STT 결과 텍스트 |
| lat | double | N | 기준 위도 |
| lng | double | N | 기준 경도 |

#### Request Example

```json
{
  "recognizedText": "부산시민공원",
  "lat": 35.1686,
  "lng": 129.0576
}
```

### Response

- HTTP Status: `200 OK`

| 필드명 | 타입 | Null 가능 | 설명 |
| --- | --- | --- | --- |
| keyword | string | N | 음성 인식 후 검색에 사용한 키워드 |
| places | array | N | 검색 결과 목록 |

#### Success Example

```json
{
  "success": true,
  "data": {
    "keyword": "부산시민공원",
    "places": [
      {
        "placeId": 10,
        "providerPlaceId": "123456789",
        "name": "부산시민공원",
        "category": "TOURIST_SPOT",
        "address": "부산광역시 부산진구 시민공원로 73",
        "point": {
          "lat": 35.1686,
          "lng": 129.0576
        },
        "matched": true
      }
    ]
  },
  "message": null
}
```

### Error

| HTTP Status | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 400 | INVALID_INPUT | 잘못된 입력입니다. | `recognizedText` 누락 또는 빈 문자열 |
| 502 | EXTERNAL_API_ERROR | 외부 API 호출에 실패했습니다. | 카카오 API 호출 실패 |
